Configurar repositorio GitHub
1
Ve a github.com/new y crea un repositorio llamado exactamente pid-plantillas. Márcalo como Público.
2
Sube tus archivos .xlsx directamente al repo. Puedes hacerlo desde la interfaz web de GitHub arrastrando los archivos.
3
Haz clic en ↻ Sincronizar GitHub arriba. La app detecta los archivos automáticamente.
4
Configura el mapeo: qué prefijo de TAG usa cada plantilla. Eso es todo — no necesitas tocar código.
